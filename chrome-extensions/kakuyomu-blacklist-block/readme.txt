1. Open Chrome and type "chrome://extensions" in the address bar, then press Enter.
2. On the extensions page, enable "Developer mode" by toggling the switch in the top right corner.
3. Click the "Load unpacked" button located in the top left corner.
4. Navigate to the folder containing the "kakuyomu-blacklist-block" extension files. 
5. Select the folder and click "Open" or "Select Folder" to load the extension.
6. Read plugin instruction.
